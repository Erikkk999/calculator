# Calculator
Built an interactive calculator using JavaScript with support for keyboard input, real-time calculations, and operation chaining. Implemented custom state management, input validation, and dynamic UI updates. Enhanced user experience with responsive design.

## Features
- Keyboard support for fast input
- Live result preview while typing
- Operation chaining
- Error handling (e.g. division by zero)

## Tech Stack
- HTML
- CSS
- JavaScript

## What I Learned
- Managing application state in JavaScript
- Using event delegation for efficient DOM handling
- Handling edge cases and input validation
- Updating UI dynamically based on state
